System Architecture

How It Works

ProcureMatch AI uses a sophisticated machine learning pipeline to analyze procurement documents and detect discrepancies between PO, GRN, and Invoice quantities.

User Input

User submits PO, GRN and Invoice quantities

Backend API

Processes the input data and prepares for model prediction

AI Model

Random Forest classifier analyzes the quantities and determines match status

Result

System returns MATCH or MISMATCH with confidence percentage

Technical Details

Model Architecture

  • Random Forest Classifier
  • Trained on synthetic procurement data
  • Accuracy: 92% on test set
  • Precision: 89% for mismatch detection

Tech Stack

  • Frontend: HTML, CSS, JavaScript
  • Backend: Python Flask API
  • Machine Learning: Scikit-learn
  • Deployment: Docker, AWS
